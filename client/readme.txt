Dependencies
------------
 You need to install JRE/JDK(1.6+).

Installing
----------
 Just unzip snova-[version].zip.

Running
-------
 1) change folder to snova's root folder.
 2) run "bin/start.sh" in Terminal (bin/start.bat in Windows)
